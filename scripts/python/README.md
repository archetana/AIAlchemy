# Python Scripts

This directory contains Python scripts for various tasks in the AIAlchemy project:

## Admin and User Management
- create_custom_admin.py - Create custom admin users
- create_user_via_api.py - Create users through the API
- verify_admin_user.py - Verify admin user setup

## Testing and Validation
- deployment_check.py - Check deployment status
- run_all_tests.py - Run all test suites
- test_nvidia_nim.py - Test NVIDIA NIM integration
- test_upload_config.py - Test upload configuration
- test_vector_setup.py - Test vector database setup
- validate_about_dialog.py - Validate about dialog
- validate_workflows.py - Validate GitHub workflows

## Cloud Setup and Testing
- setup-gcs-complete.py - Complete GCS setup
- simple_supabase_test.py - Test Supabase integration
- test-gcs-setup-simple.py - Simple GCS setup test
- test-gcs-setup.py - Full GCS setup test
