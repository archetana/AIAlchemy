# Shell Scripts

This directory contains shell scripts for deployment, setup, and testing:

## Deployment Scripts
- deploy-gcp.sh - Deploy to Google Cloud Platform
- deploy-nginx-gateway.sh - Deploy with Nginx gateway
- deploy-with-gcs.sh - Deploy with Google Cloud Storage

## Setup Scripts
- setup-gcs-storage.sh - Set up Google Cloud Storage
- start-local-dev.bat - Start local development (Windows)
- start-local-dev.sh - Start local development (Linux/Mac)

## Testing Scripts
- test_quick.sh - Quick test suite
- test-frontend-build.sh - Test frontend build
- VERIFY-APIS.sh - Verify API endpoints
